# janken-js
## Overview
- じゃんけんを模したWebアプリです。

## Demo
URL [https://osakana0000.github.io/jankenJs/](http://osakana0000.site/janken-js/)

## Discription
- 「グー」「チョキ」「パー」のいづれかを選択すると、CPUもじゃんけんの手段を表示します。
- 「ランダム」は、プレイヤーの手段をランダムで表示します。

## Features
- 勝敗の判定は以下の通りです。
  - プレイヤーとCPUの手段が同じなら「あいこ」
  - 勝ちの組み合わせ3パターンいずれかに該当すれば「勝ち」
  - 上記以外は「負け」

## Requirement
- ES6
- Bootstrap5
